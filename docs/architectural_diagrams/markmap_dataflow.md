# 🌊 Data Physics (Flow)

## 1. Ingestion (Input)
- **Raw Sources**
  - Screenshots (Images)
  - CSV/Excel (Files)
  - API Streams (JSON)
- **Processing**
  - Normalization (Standard Schema)
  - Sanitation (PII Removal)
  - Validation (Data Quality)

## 2. Storage (Memory)
- **Hot Path** (Vector)
  - Embedding (OpenAI)
  - Indexing
  - Similarity Search
- **Cold Path** (Graph)
  - Node Creation (Campaigns)
  - Edge Creation (Relationships)
  - Property Mapping (Metrics)
- **Structured** (SQL)
  - Aggregation Tables
  - Time-series Data

## 3. Retrieval (Recall)
- **Hybrid Search**
  - Semantic (Vector) + Structural (Graph)
- **Context Window**
  - RAG Injection
  - History Replay

## 4. Inference (Thought)
- **LLM Processing**
  - Prompt Engineering
  - Reasoning Chain
  - Tool Invocation

## 5. Response (Output)
- **Synthesis**
  - Natural Language
  - JSON Data
  - Visualization Config
