# 📚 Libraries & Role (Order of Flow)

## 1. Frontend (Entry)
- **React 18** (UI Library)
- **Next.js 14** (App Framework)
- **TailwindCSS** (Styling)
- **Framer Motion** (Animations)
- **Recharts** (Data Visualization)
- **Lucide React** (Icons)
- **Axios / Fetch** (HTTP Client)

## 2. API Gateway (Ingress)
- **Uvicorn** (ASGI Server)
- **FastAPI** (Web Framework)
- **Pydantic** (Data Validation)
- **Python-Multipart** (File Uploads)
- **Starlette** (Core ASGI)

## 3. Orchestration (Control)
- **LangChain Core** (LLM Interface)
- **LangGraph** (Workflow State Machine)
- **Tenacity** (Retries/Resilience)
- **Loguru** (Structured Logging)

## 4. Intelligence (Processing)
- **OpenAI SDK** (LLM Access - GPT-4)
- **Anthropic SDK** (LLM Access - Claude)
- **NumPy** (Math Operations)
- **Pandas** (Dataframes/Analysis)

## 5. Data Access (Storage)
- **Neo4j Driver** (Graph Database)
- **DuckDB** (OLAP SQL Database)
- **SQLAlchemy** (ORM)
- **Pydantic Settings** (Config Management)
- **Redis** (Caching - Optional)

## 6. Testing & Ops (Support)
- **Pytest** (Testing)
- **Docker** (Containerization)
- **Celery** (Async Workers)
